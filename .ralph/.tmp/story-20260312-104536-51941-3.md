### US-003: Click a part region to highlight, zoom, and show detail panel
Status: in_progress
Depends on: US-002

Description:
As a user, I want to click a part region on the photo to zoom into that area and see detailed information for that part so I can quickly understand its condition and priority.

Acceptance Criteria:
- [ ] Clicking a part overlay region sets that part as 'active' in component state
- [ ] Active region is highlighted with a more prominent overlay (colored border matching condition color)
- [ ] Photo container applies a CSS transform to zoom and pan to center the active part's bounding box region (scale ~2x, clipped to container)
- [ ] Zoom/pan uses CSS transition for a smooth animation (200–300ms ease)
- [ ] Part detail panel updates immediately to show: part name, component group, brand/model, condition badge, priority badge, condition notes, and its pricing results
- [ ] Corresponding row in the parts table is highlighted (scroll into view if needed)
- [ ] Clicking the active region again, or clicking outside all regions, deactivates zoom and returns photo to normal scale
- [ ] Only one part can be active at a time
- [ ] Example: click 'Rear Derailleur' overlay → photo zooms to that area, detail panel shows Shimano Claris with 'fair' condition badge
- [ ] Negative case: rapidly clicking two different part regions → second click correctly replaces first, no visual glitch
