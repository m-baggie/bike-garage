### US-002: Cycling-themed loading animation — replace generic spinner
Status: in_progress
Depends on: US-001

Description:
As a user, I want a cycling-themed loading animation during analysis so the app feels purpose-built for cyclists.

Acceptance Criteria:
- [ ] Create a CyclingLoader component in client/src/components/CyclingLoader.jsx
- [ ] Animation is a CSS-only spinning bicycle wheel: an SVG circle with 8 spoke lines, rotated via @keyframes spin at 1.2s linear infinite
- [ ] Wheel SVG is 64x64px, stroke color uses --priority-5 blue (#1D4ED8)
- [ ] Below the wheel, text reads 'Spinning up your diagnosis…' in --text-secondary color
- [ ] CyclingLoader replaces the existing spinner on the UploadPage during analysis (the 'Analyzing…' button state)
- [ ] CyclingLoader also replaces the loading overlay on ResultsPage during reanalysis
- [ ] Component accepts an optional `message` prop to override the default text
- [ ] npm run lint passes
- [ ] Example: click 'Run Diagnosis' → wheel spins with 'Spinning up your diagnosis…' below it
- [ ] Negative case: loader disappears completely and is replaced by results once analysis completes — no ghost spinner
