### US-007: [Tree 3 — Frontend] React + Vite frontend scaffold with routing
Status: in_progress
Depends on: US-001

Description:
As a developer, I want a working React frontend with client-side routing so I can build Upload and Results pages.

Acceptance Criteria:
- [ ] Install react-router-dom: `npm install react-router-dom` in /client
- [ ] App.jsx defines routes: / → UploadPage, /results → ResultsPage
- [ ] Both UploadPage and ResultsPage are stub components that render a heading (to be fleshed out in later stories)
- [ ] Vite dev server runs on port 5173 without errors
- [ ] Global CSS reset applied (box-sizing border-box, margin 0, base font)
- [ ] npm run lint passes with no errors
- [ ] Example: navigating to / renders 'Upload Page', navigating to /results renders 'Results Page'
- [ ] Negative case: unknown route (e.g. /foo) renders a 404 'Page not found' message
