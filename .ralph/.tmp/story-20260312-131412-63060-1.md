### US-001: Fix text contrast — replace dark navy text color with legible alternative
Status: in_progress
Depends on: None

Description:
As a user, I want all body text to be clearly legible against page backgrounds so the app is comfortable to read.

Acceptance Criteria:
- [ ] Replace all instances of color #213547 in CSS and inline styles with #1e293b (slate-800) which reads clearly on white and light-gray backgrounds
- [ ] Add --text-primary: #1e293b and --text-secondary: #475569 CSS tokens to :root in index.css
- [ ] All body text, table content, labels, and headings reference --text-primary or --text-secondary
- [ ] No raw #213547 hex remains anywhere in the codebase
- [ ] npm run lint passes
- [ ] Example: parts table rows show dark legible text on white background
- [ ] Negative case: no text appears as near-invisible dark-on-dark anywhere on the results page
