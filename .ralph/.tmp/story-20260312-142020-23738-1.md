### US-006: Minimal custom scrollbars and top accent bar
Status: in_progress
Depends on: US-001

Description:
As a user, I want the scrollbars to be minimal and unobtrusive and the page to have a branded top accent so the app feels polished and cohesive.

Acceptance Criteria:
- [ ] Add global ::-webkit-scrollbar styles to index.css: width 5px, height 5px
- [ ] ::-webkit-scrollbar-track background transparent
- [ ] ::-webkit-scrollbar-thumb background rgba(100, 108, 255, 0.4), border-radius 999px
- [ ] ::-webkit-scrollbar-thumb:hover background rgba(100, 108, 255, 0.8)
- [ ] Add a 4px tall top accent bar to the app layout (above the main content, full width): background linear-gradient(90deg, var(--priority-5) 0%, var(--condition-good) 100%)
- [ ] Accent bar is fixed/sticky at the very top of the viewport, z-index above all content
- [ ] npm run lint passes
- [ ] Example: scroll the results page → thin purple-to-teal scrollbar thumb visible on the right, no default OS scrollbar styling
- [ ] Negative case: accent bar does not shift page content down by more than 4px — no layout shift on the upload or results page
